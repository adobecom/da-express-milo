MWPW-187074 - Standalone reusable toolbar for Color Explorer

Desktop/XL Toolbar (min-width: 1024px)

Features: Horizontal layout, all elements in one row, 1600px max-width, elevated shadow

Tablet/M Toolbar (min-width: 768px)

Features: Horizontal layout, 835px max-width, simplified shadow

Mobile/S Toolbar (max-width: 767px)

Features: Vertical stacked layout, rounded top corners, color swatches above

Sticky Toolbar (Viewport-Sticky on Scroll)

Features: Stays fixed at bottom of viewport when scrolling, adaptive to breakpoints

Scroll down to see the sticky toolbar

This toolbar will remain visible at the bottom of the viewport as you scroll through this content area. The sticky variant is ideal for pages where users need persistent access to palette actions while browsing content.
